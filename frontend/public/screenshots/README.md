# Screenshots Folder

Place your screenshot images here with these exact names:

1. **employee-dashboard.png** - Employee dashboard screenshot
2. **admin-dashboard.png** - Admin dashboard screenshot  
3. **attendance-reports.png** - Attendance reports page screenshot
4. **security-logs.png** - Security logs page screenshot

## Image Guidelines:
- Format: PNG or JPG
- Recommended size: 1200x800 pixels
- File size: Less than 500KB each
- Use high quality screenshots
- Blur any sensitive data before uploading

If images are not found, placeholder colored boxes will be shown instead.
